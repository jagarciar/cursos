¿Cómo crear un proyecto AngularJS?

Después de instalar Angular JS podemos realizar la creación de un proyecto ejecutando un comando npm como se muestra a continuación:

ng new project-name

Debe reemplazar project-name por el nombre del proyecto.

Al ejecutar el comando le solicitará seleccionar un framework CSS. Podemos seleccionar el por defecto.

Después le preguntará si desea activar Server-Side Rendering (SSR). SSR es un enfoque en la cuál las páginas web se generan en el servidor antes de ser enviadas al navegador. En otras palabras, el servidor procesa la lógica y la estructura de la página y envía la página completamente renderizada al navegador del usuario.

A continuación nos solicitará seleccionar algún agente AI. Podemos dejar la selección por defecto.

Finalmente creará el proyecto.
